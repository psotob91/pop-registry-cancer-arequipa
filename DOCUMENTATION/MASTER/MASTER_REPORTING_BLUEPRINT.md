# MASTER_REPORTING_BLUEPRINT

## 1. Informe de resultados

### Secciones mínimas
1. Resumen ejecutivo
2. Métodos
3. Resultados
4. Discusión breve operativa
5. Limitaciones
6. Anexos

## 2. Tablas obligatorias
- casos por año y sexo
- casos por grupos amplios de edad
- top 10 cánceres por sexo
- tasas crudas del periodo
- tasas ajustadas del periodo
- tablas anuales si se activa `denominator_mode = annual`

## 3. Gráficos obligatorios
- distribución por edad
- tendencia anual 2015–2022
- top cánceres
- gráfico de base diagnóstica
- gráficos de calidad seleccionados

## 4. Informe de calidad
Debe incluir como mínimo:
- MV %
- DCO %
- edad desconocida %
- sexo desconocido %
- residencia desconocida %
- consistencia vital
- tabla de decisiones metodológicas adoptadas

## 5. Anexos obligatorios
- diccionarios locales
- trazabilidad de reglas
- tablas completas por edad quinquenal
- reglas de denominadores y estandarización

## 6. Texto automático reusable
Ejemplos:
- “Para el periodo 2015–2022 se adoptó como estrategia principal el uso de población de mitad de periodo (año 2018), manteniendo una alternativa anual parametrizable.”
- “La base diagnóstica se clasificó a partir del código local BASE #7 con agrupaciones analíticas explícitas y trazables.”
